# AWS Scripts

This directory contains scripts to interact with AWS services, particularly for
managing EC2 instances. These scripts can help automate tasks such as connecting
to instances, managing security groups, and deploying applications.

**NOTE:** These scripts are provided as references and will not work with
student AWS accounts due to permission restrictions.

## Scripts

- `ec2-launch.sh`: A script to launch a new EC2 instance with predefined settings.
- `ec2-terminate.sh`: A script to terminate an existing EC2 instance.
- `ssh-ec2.sh`: A script to connect to an existing EC2 instance via SSH.
